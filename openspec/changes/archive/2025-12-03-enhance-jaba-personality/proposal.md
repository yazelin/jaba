# Proposal: enhance-jaba-personality

## Summary
增強呷爸的個性設定、健康飲食提醒與卡路里估算功能，以及管理員店家建議功能。

## Why
目前呷爸的 prompts 需要調整：
1. 個性設定不夠明確 - 應該是「成熟穩重的大叔」形象
2. 缺少健康飲食提醒功能 - 對點餐使用者推薦較健康的餐點
3. 缺少店家建議功能 - 對管理員提供今日店家選擇建議（基於歷史訂餐記錄）

## What Changes

### 1. 調整呷爸個性（prompts）
- 成熟穩重的大叔形象
- 語氣溫和但不過度熱情
- 說話有條理、給人信賴感

### 2. 健康飲食提醒與卡路里估算（user_prompt + 訂單資料 + UI）
- 當使用者點餐時，適時推薦較健康的選項
- 提醒均衡飲食的重要性
- 不強制，只是友善建議
- **估算餐點卡路里**：點餐時估算每道餐點的大約卡路里
- **儲存卡路里至訂單**：將每個品項的估算卡路里存入訂單資料
- **UI 顯示卡路里**：在「我的訂單」頁面顯示各餐點卡路里和總卡路里

### 3. 店家建議功能（manager_prompt + context）
- 當今日尚未設定店家時，分析過去幾天的訂餐記錄
- 建議今天可以換一家店
- 如果今日店家已設定，則不主動建議

## Scope
- 修改 `data/system/prompts/jaba.json` 的 `user_prompt` 和 `manager_prompt`
- 修改 `app/claude.py` 的 `build_context()` 為管理員提供歷史店家資訊
- 修改 `app/claude.py` 的 `_create_order()` 儲存卡路里資訊
- 修改前端 UI 顯示卡路里資訊
