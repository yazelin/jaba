# redesign-order-page-layout

## Summary
將訂餐頁面改為三欄式佈局（與管理頁一致），並調整團隊聊天室預設為展開狀態。

## Motivation
目前訂餐頁是兩欄佈局（左對話、右側邊欄），所有面板擠在右側較窄的空間。使用者需求：
- 左側：今日店家、菜單（方便點餐時參考）
- 中間：AI 對話框
- 右側：我的訂單、大家的訂單（訂單相關資訊）
- 團隊聊天室預設展開（目前預設收合）

這樣的佈局讓使用者在點餐時更容易參考菜單，同時能看到自己和大家的訂單。

## Scope
- **範圍內**：
  - `templates/order.html` 結構調整為三欄
  - `static/css/style.css` 新增/調整三欄樣式
  - 新增「大家的訂單」面板
  - 團隊聊天室預設展開
- **範圍外**：管理頁面、首頁維持不變

## Dependencies
- 依賴 `show-user-own-orders`（已完成）：「我的訂單」過濾邏輯
