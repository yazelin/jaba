# Proposal: fix-mobile-ui-overlap

## Why

手機版 UI 有兩個視覺問題影響使用體驗：

1. **美食評論區遮擋內容**
   - 訂餐頁：美食評論區 (position: fixed, bottom: 10px) 遮擋「大家的訂單」面板底部的「共 x 筆」統計行，使用者無法在不捲動的情況下看到訂單總數
   - 管理員頁：同樣遮擋收款面板的下緣

2. **返回看板按鈕不明顯**
   - 目前是透明背景的文字連結，容易被忽略
   - 使用者不知道如何返回首頁

## What Changes

1. **增加頁面底部留白**：在手機版為主要內容區增加 padding-bottom，讓美食評論區不會遮擋重要資訊

2. **重新設計返回按鈕**：將「← 返回看板」改為有背景的按鈕樣式，更像可點擊的元素

## Scope

- ui-layout spec
